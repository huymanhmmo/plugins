# AI Auto Content Writer

Plugin WordPress hỗ trợ tự động viết bài chuẩn SEO, tối ưu hình ảnh, lên lịch đăng bài nhờ AI (Google Gemini và các model AI miễn phí)!

## Các tính năng chính:
- Quản lý API AI và nguồn ảnh
- Lưu và duyệt nguồn tham khảo từ web
- Tạo bài viết bằng AI, chọn style và cấu hình chuẩn SEO
- Lên lịch bài đăng tự động
- Chèn ảnh đại diện, tối ưu ảnh chuẩn SEO
- Nhật ký hoạt động
- Dễ dàng cài đặt và mở rộng

## Hướng dẫn sử dụng:
1. Upload vào wp-content/plugins và kích hoạt
2. Cấu hình API và nguồn ảnh ở menu plugin
3. Thêm nguồn tham khảo, cấu hình AI
4. Tạo bài mới hoặc lên lịch tự động
5. Xem nhật ký hoạt động
